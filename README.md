# Unit Testing

This repository contains **unit testing exercises and examples** for QA purposes. The projects focus on writing and practicing unit tests in C#.

---

## Repository Contents

- **Unit Testing Arrays/** – Exercises for testing array operations.  
- **Unit Testing Lists/** – Exercises for testing list operations and collection handling.  
- **Unit Testing Methods/** – Exercises for testing methods and functions with various inputs.  

---

## Purpose

The purpose of this repository is to:

- Practice writing unit tests for different data structures and methods  
- Understand best practices for test-driven development (TDD)  
- Improve programming skills in the context of QA automation  
- Organize unit testing exercises in a structured way  

---

## How to Use

1. Clone or download the repository.  
2. Open the project folders in **Visual Studio** or any compatible IDE.  
3. Explore the unit test files and run the tests using **NUnit Test Explorer** or the command line:

```bash
dotnet test
```

---

## Notes

- Each folder contains exercises for testing specific types of data or methods.  
- These exercises are designed for learning and practicing unit testing techniques.  
- You can modify and extend the tests to explore additional scenarios and edge cases.
